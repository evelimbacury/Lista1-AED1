Comandos do git para usar:

git push -u origin master -> Trazer as atualizações para o repositório remoto (enviar para o Github)

git status -> Ver o estado das atividades no seu repositório

git add . -> Atualizar as atividades e adicioná-las para ser enviadas

git commit -m "mensagem" ->

git pull origin master -> Baixar as atualizações remotas para seu computador (baixar os arquivos alterados do GitHub)
